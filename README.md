# PromptMap

A visual, interactive prompt design engine for Claude.

PromptMap guides you from a rough idea through intelligent, sequential questions until your idea is clear and structured enough to generate an expert-level Claude prompt.

## What it does

- Start with any rough idea — a product, strategy question, creative direction, or anything else
- PromptMap infers the prompt destination (Build / Analyse / Write / Design / Research / Strategise)
- Sequential questions branch into a visual mind map
- A live Prompt Arc narrates how your idea is taking shape
- Calibration actions let you steer, deepen, or redirect at any point
- Generates a copy-paste-ready Claude prompt from the full journey

## Run locally

Open `index.html` in any browser. No build step, no dependencies.

## Deploy

Connected to Vercel — pushes to `main` deploy automatically.
